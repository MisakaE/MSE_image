# 使用RUST编写的一款轻量级图片查看器

### 使用方法:

~~~
msev {image_path} //仅打开目标图片

msev -d {dir_path} //打开目标文件夹
~~~
进入程序后的按键
~~~
a/d/w/s :控制图片位置
q/e     :放大/缩小图片
z/c     :切换上/下一张图片（当使用-d打开文件夹时可用）
~~~
